# Webpack-multi-page

Multi page development environment based on webpack

<code>node v8.9.1</code> &nbsp; <code>yarn v1.7.0</code> &nbsp; <code>OS Windows 10</code>

#### Build Setup

``` bash
# install dependencies
yarn install

# serve with hot reload at 192.168.1.6:9000
yarn run dev

# build
yarn run build

```
